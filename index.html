杂项-PM2相关命令

pm2 相关

pm2需要全局安装

进入项目根目录

2.1 启动进程/应用 pm2 start bin/www 或 pm2 start app.js

2.2 重命名进程/应用 pm2 start app.js --name wb123

2.3 添加进程/应用 watch pm2 start bin/www --watch

2.4 结束进程/应用 pm2 stop www

2.5 结束所有进程/应用 pm2 stop all

2.6 删除进程/应用 pm2 delete www

2.7 删除所有进程/应用 pm2 delete all

2.8 列出所有进程/应用 pm2 list

2.9 查看某个进程/应用具体情况 pm2 describe www

2.10 查看进程/应用的资源消耗情况 pm2 monit

2.11 查看pm2的日志 pm2 logs

2.12 若要查看某个进程/应用的日志,使用 pm2 logs www

2.13 重新启动进程/应用 pm2 restart www

2.14 重新启动所有进程/应用 pm2 restart all

创建一个项目

基于vue-cli搭建一个全新的项目

vue-cli
创建一个项目

安装@vue/cli

  1. 使用下列命令安装@vue/cli

    本人第一次安装未成功, 如下:

    提示permission denie，由此可知大概率是权限的问题，因此修改安装命令为 sudo npm install -g @vue/cli。执行后看到如下信息，表示安装成功。

  2. 打开terminal, 输入vue --version命令检查vue-cli是否安装成功

创建一个项目

  1. 使用 vue create hello-world 命令创建一个vue前端工程, 此时会让你选择默认的preset或者手动配置，快速生成一个项目可以选择babel+eslint

  2. 选择可preset后就是等待时间了，等命令执行完成，出现了 Successfully created project hello-world.,代表工程已经创建完成。

  3. 根据上图中提示，输入命令cd hello-world 、yarn serve启动, 然后浏览器打开 http://localhost:8080 ，此时你的第一个前端工程就呈现在面前了。

将项目添加到vscode中

  1. 启动vscode应用程序，然后然后根据“文件(File)->将文件夹添加到工作区(Add Folder To Workspace)”，打开选择框，选择fork下来的工程目录。

一个初始的vue框架工程就形成了，你可以在此基础上愉快的码了。
